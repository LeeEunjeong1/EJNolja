## EJNolja
전국 관광정보를 모아서 보여주는 앱
## 기술 스택 & 라이브러리
- IDE : Android Studio
- Language : Kotlin
- Network  : Retrofit 
- Android Jetpack(dataBinding, ViewBinding, RecyclerView, ViewModel, LiveData)
- MVVM Pattern

## 기능
- 로그인 / 회원가입 (Retrofit통신을 이용한 RestAPI 구현)
- 자동로그인 (SharedPreferences)
- 관광 정보 무한 스크롤 (RecyclerView)
- 지역별 광광지 모아보기 (Spinner)
- 지역 검색기능
