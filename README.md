# EJNolja
전국 관광정보를 모아서 보여주는 앱 
- Android, Kotlin
- MVVM
### 주요 기능 및 사용 기술
- Retrofit통신을 이용한 RestAPI구현 -> 로그인, 회원가입 기능
- SharedPrefrerence 이용하여 자동로그인 구현
- RecyclerView
- Spinner처리를 통한 지역별 관광지 검색
